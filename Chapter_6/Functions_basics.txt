What are functions ??

Functions in Python are the group of reusable code into a block that can be executed by calling the function name. This helps in avoid of repetition and makes programs modular and readable.

There are many in-build functions in python like print(), input(), len(), etc.

But we can create our own function and they are called as user defined functions.

To mke your own function we have to use "def" keyword and then name the function. After this we have to call the function name() and paranthesis.



==> Syntax :-

def greet():                #function declaration
     print("Hello, welcome to Python!")

greet()                     #Calling the function



==> What are parameters in function ?
==> Parameters are variables listed inside the function definition and are created inside the function paranthesis ().
